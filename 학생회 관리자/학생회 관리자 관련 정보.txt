학생회 관리자 설명
studentAdmin.jsp : 학생회 관리자 메인 화면studentBazaarAdmin.jsp : 바자회 회차별 화면(메뉴에서 바자회 관리에서 회차별로 누르면 나오는 화면)studentBazzarEnroll.jsp : 바자회 등록 화면studentBazzarInfoAdmin.jsp : 메뉴에서 바자회 관리 누르면 나오는 화면(바자회 정보 화면)studentBazzarInfoUpdateAdmin.jsp : 바자회 정보 화면에서 연필 버튼 누르면 나오는 바자회 정보 수정 화면studentCountAdmin.jsp : 메뉴에서 도서/대여관리에서 재고관리 누르면 나오는 화면studentDamageAdmin.jsp : 메뉴에서 도서/대여관리에서 훼손관리 누르면 나오는 화면

<c:forEach items="${bookData}" var="bookVO">
  <div class="cardee ripple">
      <c:choose>
            <c:when test="${bookVO.book_enroll_condition == 0}">
            </c:when>
            <c:when test="${bookVO.book_enroll_condition == 1}">
               <span class="badg badge-ing">대여중</span>
            </c:when>
            <c:when test="${bookVO.book_enroll_condition == 2}">
               <span class="badg badge-rec">예약중</span>
            </c:when>
            <c:when test="${bookVO.book_enroll_condition == 3}">
               <span class="badg badge-danger">훼손</span>
            </c:when>
      </c:choose>
      <c:choose>
         <c:when test="${bookVO.book_enroll_condition == 3}">
            <img class="finish__image" src="img/${bookVO.book_img}" height="300" />
         </c:when>
         <c:otherwise>
        	<a href="Controller?cmd=rentalBookAction&number=${bookVO.book_num}">
           		<img src="img/${bookVO.book_img}" height="300" />
         	</a>
         </c:otherwise>
      </c:choose>
     <input type="hidden" name="num" value="${bookVO.book_num}">
     <div class="pname">
        <c:out value="${bookVO.book_name}" />
     </div>
  </div>
</c:forEach>
